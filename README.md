# Snowflake1D

## Setup Instructions
<u> Create Virtual Environment</u>
<br>
<br>
(powershell) <br>
``python -m venv .\venv `` <br>

``.\venv\Scripts\activate`` <br>
<br>
<u> Install Requirements</u>

``pip install -r requirements.txt`` <br>
<br>
<u> Run the App Locally</u>

``python -m streamlit run app.py`` <br>
