# Snowflake1D

## Setup Instructions
<b> Create Virtual Environment</b>
<br>
(powershell) <br>
``python -m venv .\venv `` <br>

``.\venv\Scripts\activate`` <br>
<br>
<b> Install Requirements</b>

``pip install -r requirements.txt`` <br>
<br>
<b> Run the App Locally</b>

``python -m streamlit run app.py`` <br>
